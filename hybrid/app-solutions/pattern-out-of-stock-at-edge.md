## <a name="solution"></a><span data-ttu-id="16afe-109">Lösung</span><span class="sxs-lookup"><span data-stu-id="16afe-109">Solution</span></span>

<span data-ttu-id="16afe-110">Im Lösungsbeispiel wird ein Edgegerät, wie beispielsweise Azure Stack Edge, in jedem Geschäft verwendet, das Daten von Kameras im Geschäft effizient verarbeitet.</span><span class="sxs-lookup"><span data-stu-id="16afe-110">The solution example uses an edge device, like an Azure Stack Edge in each store, which efficiently processes data from cameras in the store.</span></span> <span data-ttu-id="16afe-111">Dieses optimierte Design ermöglicht es Händlern, nur relevante Ereignisse und Bilder in die Cloud zu senden.</span><span class="sxs-lookup"><span data-stu-id="16afe-111">This optimized design lets stores send only relevant events and images to the cloud.</span></span> <span data-ttu-id="16afe-112">Das Design spart Bandbreite und Speicherplatz und stellt den Schutz von Kundendaten sicher.</span><span class="sxs-lookup"><span data-stu-id="16afe-112">The design saves bandwidth, storage space, and ensures customer privacy.</span></span> <span data-ttu-id="16afe-113">Während Frames von jeder Kamera gelesen werden, verarbeitet ein ML-Modell das Bild und gibt alle Bereiche mit Fehlbeständen zurück.</span><span class="sxs-lookup"><span data-stu-id="16afe-113">As frames are read from each camera, an ML model processes the image and returns any out of stock areas.</span></span> <span data-ttu-id="16afe-114">Das Bild und die Bereiche mit Fehlbeständen werden in einer lokalen Web-App angezeigt.</span><span class="sxs-lookup"><span data-stu-id="16afe-114">The image and out of stock areas are displayed on a local web app.</span></span> <span data-ttu-id="16afe-115">Diese Daten können an eine Time Series Insight-Umgebung gesendet werden, um in Power BI Erkenntnisse anzuzeigen.</span><span class="sxs-lookup"><span data-stu-id="16afe-115">This data can be sent to a Time Series Insight environment to show insights in Power BI.</span></span>

<span data-ttu-id="16afe-117">Die Lösung funktioniert so:</span><span class="sxs-lookup"><span data-stu-id="16afe-117">Here's how the solution works:</span></span>

1. <span data-ttu-id="16afe-118">Bilder werden von einer Netzwerkkamera über HTTP oder RTSP aufgenommen.</span><span class="sxs-lookup"><span data-stu-id="16afe-118">Images are captured from a network camera over HTTP or RTSP.</span></span>
2. <span data-ttu-id="16afe-119">Die Größe des Bilds wird geändert und zum Rückschlusstreiber gesendet, der mit dem ML-Modell kommuniziert, um festzustellen, ob es Bilder zu einem Fehlbestand gibt.</span><span class="sxs-lookup"><span data-stu-id="16afe-119">The image is resized and sent to the inference driver, which communicates with the ML model to determine if there are any out of stock images.</span></span>
3. <span data-ttu-id="16afe-120">Das ML-Modell gibt alle Bereiche mit Fehlbeständen zurück.</span><span class="sxs-lookup"><span data-stu-id="16afe-120">The ML model returns any out of stock areas.</span></span>
4. <span data-ttu-id="16afe-121">Der Rückschlusstreiber lädt das unbearbeitete Bild in ein Blob hoch (sofern angegeben) und sendet die Ergebnisse aus dem Modell an Azure IoT Hub und einen Begrenzungsrahmenprozessor auf dem Gerät.</span><span class="sxs-lookup"><span data-stu-id="16afe-121">The inferencing driver uploads the raw image to a blob (if specified), and sends the results from the model to Azure IoT Hub and a bounding box processor on the device.</span></span>

6. <span data-ttu-id="16afe-123">Die Web-App fragt Bildern ab und zeigt sie in der Reihenfolge des Empfangs.</span><span class="sxs-lookup"><span data-stu-id="16afe-123">The web app queries for images and shows them in the order received.</span></span>
7. <span data-ttu-id="16afe-124">Nachrichten von IoT Hub werden in Time Series Insights aggregiert.</span><span class="sxs-lookup"><span data-stu-id="16afe-124">Messages from IoT Hub are aggregated in Time Series Insights.</span></span>
8. <span data-ttu-id="16afe-125">Power BI zeigt einen interaktiven Bericht zu Artikeln mit Fehlbeständen im Zeitverlauf mit den Daten aus Time Series Insights an.</span><span class="sxs-lookup"><span data-stu-id="16afe-125">Power BI displays an interactive report of out of stock items over time with the data from Time Series Insights.</span></span>
